# Security Toolbox

A collection of practice security utilities built with Python.

## Goals

- Learn Python
- Practice Git & Github
- Build reusable security tools
- Grow into an AI-powered security platform

## Planned Features

- Password Generator
- SHA256 Hash
- Base64 Encode / Decode
- UUID Generator
- URL Encode / Decode
- JWT Decoder
- Port Checker
